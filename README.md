# Machine Learning Internship Projects

## Overview

This repository contains four machine learning tasks completed as part of my internship. The projects demonstrate skills in data preprocessing, model building, deep learning, and recommendation systems using Python and popular ML libraries.


##Tasks Completed

### Task 1 — Decision Tree Classification

Built a Decision Tree model to classify data based on input features.

**Key Steps**

* Data loading and preprocessing
* Feature selection
* Model training using Scikit-learn
* Model evaluation

**Skills Used**
Python, Pandas, Scikit-learn, Data Visualization


### Task 2 — Sentiment Analysis

Performed sentiment classification on text reviews.

**Key Steps**

* Text preprocessing
* Tokenization
* Feature extraction
* Model training

**Skills Used**
Natural Language Processing, Pandas, Scikit-learn


### Task 3 — CNN Image Classification

Built a Convolutional Neural Network model to classify images using deep learning.

**Key Steps**

* Data normalization
* CNN model architecture
* Training and validation
* Performance evaluation

**Skills Used**
TensorFlow, Keras, Deep Learning

### Task 4 — Recommendation System

Created a recommendation system based on user ratings.

**Key Steps**

* Data preprocessing
* Similarity calculation
* Generating recommendations

**Skills Used**
Python, Pandas, Collaborative Filtering

##  Installation

1️⃣ Clone the repository
git clone <your-repo-link>


2️⃣ Navigate to project folder
cd Machine_Learning_Internship

3️⃣ Create virtual environment

python -m venv venv
4️⃣ Activate environment

Windows:
venv\Scripts\activate

5️⃣ Install dependencies
pip install -r requirements.txt

## How to Run

Open the notebooks in VS Code or Jupyter and run all cells.
